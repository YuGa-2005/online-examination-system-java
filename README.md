# Online Examination System (Java)

## About the Project
This is a console-based Online Examination System developed using Core Java
as part of my Virtual Internship at Oasis Infobyte. The project allows users
to log in, update their profile, attempt MCQ-based exams with a timer, and
automatically submit answers.

## Features
- Login and Logout System
- Update Profile and Password
- MCQ-Based Online Examination
- Timer with Auto Submit
- Result Calculation

## Technologies Used
- Java (Core Java)
- Object-Oriented Programming
- Arrays and Control Statements
- Scanner Class

## Project Structure
Main.java        - Application entry point  
User.java        - User authentication and profile management  
Exam.java        - Exam logic and timer implementation  
Question.java   - MCQ question model

javac *.java
java Main
